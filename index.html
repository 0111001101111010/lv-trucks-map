About the mobile food vendor program

The intent of this pilot program is to allow Mobile Food Vehicles to do business in the downtown area. It provides designated parking spaces within the public right of way and ALSO will allow food trucks to open for periods of time longer than 30 minutes. This program does not apply to special events. Supporting ordinances for this program have been passed by the City Council.

About this site

This site was built by Code for America for the City of Las Vegas. It is supposed to get some information via the Parkeon meter API to provide real-time information about truck locations.
